上传代码到码云（github同）
发表于 2020-04-27| 更新于 2020-04-27|github
字数总计:228|阅读时长: 1 分钟
|阅读量:

1、安装git客户端，安装过程简单，省略。

2、git配置

git config --global user.name "用户名"
git config --global user.email "邮箱"

2、生成公钥私钥

ssh-keygen -t rsa -C "邮箱"

然后会有几个确认提示，一直回车即可，完成后，在.ssh/id_rsa.pub里面就会有生成的公钥

3、在码云上建立git仓库，在项目文件夹下执行git命令

git init

4、将项目添加到仓库中

git add .

5、将添加的项目提交到仓库

git commit -m '备注'

6、将本地仓库关联到码云上

git remote add origin "自己的仓库地址"

7、上传码云前要先pull下

git pull origin  master

8、最后一步，上传代码到码云仓库

git push -u origin master

如果这一步报错，则回到上一步，重新执行命令，会下载README.md文件到本地，然后就可以正常上传了

git pull --rebase origin master
